Benutzerverwaltung

Bearbeiten Sie zunächst die Schüler- und Lehrerlisten. Bearbeiten Sie zunächst die Schülerlisten.

Mit der Überprüfung der Benutzerlisten werden die Daten zur Übertragung ins System vorbereitet.

Auf der Seite zur Übernahme der Benutzerdaten können Sie die Übertragung ins System starten und sich Informationen zum Prozess anzeigen lassen.

Info

Beachten Sie, dass die Bearbeitung der Dateien extraklassen.txt, extrakurse.txt und sperrklassen.txt nicht in der Schulkonsole abgebildet ist.

Alle sonstigen Features der Benutzerverwaltung sophomorix können Sie über die Schulkonsole nutzen.
