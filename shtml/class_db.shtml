Klassen :: Datenbanken

Ausgewählte Klasse:

Für diese Klasse wurden Datenbanken eingerichtet.

Für die Klasse gibt es keine Datenbanken.

Hier können Sie MySQL-Datenbanken für alle Schüler/innen der Klasse anlegen oder wieder löschen.

Kommen neue Schüler/innen oder Lehrer/innen zur Klasse hinzu, so können Sie über die Schaltfläche Datenbanken anlegen/aktualisieren Datenbanken für die neuen Klassenmitglieder anlegen lassen.

Die Zugangsdaten zur Datenbank werden an alle Klassenmitglieder per E-Mail verschickt.
